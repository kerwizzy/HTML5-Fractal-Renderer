Fractals

Fractals are shapes that include smaller features which resemble the fractal as a whole. These smaller shapes, in turn, include yet smaller "copies" of the larger figure.

The Mandelbrot set is a fractal because it includes smaller shapes that look like the larger shape. Interestingly, though, the smaller features in the Mandelbrot set are always slightly different than the larger fractal.

The Mandelbrot set (and the other fractals displayed here) are made by "iterating" functions on the location.

The simple explanation is like this. Each position in the picture represents a number in the complex plane. The number is placed in a function, and then the function is repeated over and over again. If the number grows larger and larger and larger, then the starting number is not in the "set". If, however, the number does not get increasing larger, then it is in the set and is colored black.

The other colors (for things not in the set) are created by showing how long it took for the number to go a certain distance.

About the Mandelbrot Set

The Mandelbrot Set is made by "iterating" the equation zn+1 = zn2 + c

Where

  • c is the complex number
  • zn is the nth number in the sequence. (the current number)
  • zn+1 is the next number in the sequence.

So, starting from z0 as 0, one can keep running this function indefinetly. Some number get larger and larger. For instance, the putting 1 in for c gives the sequence:

0, 1, 2, 5, 26...

That means 1 is not in the Mandelbrot "set". However, putting in -1 for c gives the sequence:

0, -1, 0, -1, 0...

since this sequence repeats (and thus does not get larger and larger) -1 is in the Mandelbrot set! To learn more about the Mandelbrot Set, take a look at the Wikipedia Article!

About the Burning Ship Fractal

The burning ship fractal is closely related to the Mandelbrot set. The only difference is that the real and imaginary parts of the number are changed to thier absolute values (that is, if they are negative, they are changed to positive) before each iteration. To learn more about the Burning Ship Fractal, take a look at the Wikipedia Article!

About the "Tricorn-Mandelbrot" fractal

The author discovered this fractal when experimenting with the iteration function inside the program. It is identical to the Mandelbrot Set calculation, execpt the real and imaginary terms are reversed before each iteration.

It is called the "Tricorn-Mandelbrot" fractal because it has three corners, like a tricorn-hat, and it has features that are very similar to the Mandelbrot Set.
